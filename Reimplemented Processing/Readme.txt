1. The Zip file contains war file Server project (SWE642Assignment4), UI project(UIModuleSWE642Assignment4)
2. Import SWE642Assignment4 in your eclipse
3. mvn install:install-file -Dfile=<path>/ojdbc7.jar -DgroupId=com.oracle -DartifactId=ojdbc7 -Dversion=12.2.0.1 -Dpackaging=jar
4. In UI project,please run command: npm install from folder UIModuleSWE642Assignment3.
5. run the UI server using: ng serve --open
6. Please make sure you are connected to GMU DB.
